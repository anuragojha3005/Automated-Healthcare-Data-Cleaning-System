# 🏥 Automated Healthcare Data Cleaning & Validation System

## 📌 Overview

The **Automated Healthcare Data Cleaning & Validation System** is a Python-based data quality pipeline that automates the profiling, cleaning, validation, and reporting of healthcare datasets. It helps improve data quality before datasets are used for analytics, reporting, or machine learning.

The project is divided into independent modules, making it scalable, maintainable, and easy to extend.

---

# 🚀 Project Modules

## 📊 Module 1 – Advanced Data Profiling & Metadata Intelligence

This module analyzes raw datasets and generates comprehensive profiling reports.

### Features
- Metadata Extraction
- Data Type Inference
- Semantic Detection
- PII Detection
- Missing Value Analysis
- Duplicate Detection
- Cardinality Analysis
- Rule Engine
- Profiling Report Generation
- Data Quality Visualizations

📄 Documentation: `module1_profiling/README.md`

---

## 🧹 Module 2 – Automated Data Cleaning & Validation

This module cleans and validates datasets using the profiling results generated by Module 1.

### Features
- Missing Value Handling
- Duplicate Removal
- Data Type Conversion
- Text Standardization
- Email & Phone Validation
- Data Quality Score
- Cleaning Report
- Transformation Log
- Dashboard Summary

📄 Documentation: `module2_cleaning/README.md`

# ▶️ Installation

```bash

pip install -r requirements.txt
```
